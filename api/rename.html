📂 파일명 일괄 변경

명단표(번호+이름)에 맞춰 PDF 파일명을 "번호 이름" 형식으로 한 번에 바꿔드려요.
STEP 1

명단표 업로드

번호와 이름이 들어있는 엑셀(.xlsx) 또는 CSV 파일을 올려주세요. 첫 번째 열은 번호, 두 번째 열은 이름으로 인식합니다.
📋 클릭 또는 드래그하여 명단표 업로드
STEP 2

PDF 파일 업로드

파일명에 학생 번호가 포함된 PDF들을 올려주세요. 자동으로 명단표와 매칭해 드려요.
새 파일명 형식 (자유롭게 수정 가능 — {번호}, {이름}, {원본파일명} 사용 가능)
📂 PDF 파일들을 끌어다 놓거나 클릭하여 선택 (여러 개 가능)
STEP 3

매칭 확인 & 다운로드

매칭이 잘못됐거나 비어있는 칸은 새 파일명 입력란을 직접 수정하면 돼요.
